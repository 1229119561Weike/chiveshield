    Provider 边界

    支付、搜索、生成都要可追溯；demo/mock 不能装作真实发生。

    PaymentProvider

    createPaymentLink(order) 返回支付链接；真实托管版使用 Matrix receivable，metadata 必须包含 order_id。

    ReportGenerationProvider

    generate(order, demand) 在订单 paid 后触发；本地 demo 使用内置规则，托管版使用 Matrix 模型与搜索能力。

    SearchProvider

    search({ queries, serviceType, locale }) 返回可追溯资料；没有 live provider 时只允许 DemoProvider。
